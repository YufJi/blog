# 步骤一：实现createElement方法

我们应该都知道jsx这种语法糖，都会被转换成标准的js，例如：

// input
const element = (
  <div className="cls">
    <text>hello</text>
  </div>
)

// output
const element = React.createElement(
  'div',
  {
    className: 'cls'
  },
  React.createElement('text', null, 'hello')
)
